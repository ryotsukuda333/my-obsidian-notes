# 4. 結論（わかりやすく）

### ✅ **基本的にすべて「安全圏」**

耐容上限量（UL）の 30〜80%以下に収まっている。

### ⚠ **唯一注意が必要なのはマグネシウム**

- 食事摂取基準では「サプリ由来のマグネシウム UL = 350mg/日」
    
- あなたの摂取量 ＝ **314 mg（ほぼ上限）**
    

→ ただし「下痢以外の副作用はほぼない」  
→ 食事で摂るマグネシウムは UL対象外なので問題なし。

### 🎯 **食事でさらに摂ってよい目安**

上限まで安全に追加できる量：

|栄養素|食事で追加できる量|
|---|---|
|ビタミンA|+2000 μg 程度安全|
|ビタミンD|+90 μg まで安全|
|ビタミンB群|基本無制限（尿で排泄）|
|亜鉛|+35 mg|
|セレン|+400 μg|
|カルシウム|+2000 mg|
|マグネシウム|**食事分なら無制限（UL対象外）**|
